# Report Template

All rendered reports must use this section order:

1. Executive Summary
2. Top Findings
3. Eval Summary
4. Controls Detected
5. Coverage Statement
6. Blind Spots

## Executive Summary Fields

- Target
- Framework
- Adapter status
- Adapter mode
- Risk tier
- Decision
- Finding count
- Controls present
- Eval total, passed, failed, pass rate, and dataset hash when available

## Finding Fields

- Finding ID
- Severity
- Status
- Control ID and name
- Requirement
- Location
- Recommendation
- Record hash

## Excel Sheets

- `Summary`
- `Findings`
- `Controls`
- `Eval`

## DOCX Sections

- Title
- Executive Summary
- Top Findings
- Coverage Statement

DOCX is intentionally concise. Use Markdown or Excel for detailed machine-readable review.
